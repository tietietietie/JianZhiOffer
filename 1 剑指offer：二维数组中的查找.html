问题描述：一个特殊的整数二维数组，每行递增，每列递增，查找给定值

Solution 1：暴力法
public class Solution {
    public boolean Find(int target, int [][] array) {
        for(int i = 0; i < array.length; i++)
            for(int j = 0; j < array[0].length; j++)
                if(array[i][j] == target) return true;
        return false;
    }
}
运行时间：214ms
占用内存：17556k
Idea 1
i < array.length; 不是array.length-1哦，为啥会范幼稚的错误呢。。
Problem 1
没有利用行递增和列递增的特点

Solution 2：从左下角的元素开始比较每次可以减少一行或者一列元素。
public class Solution {
    public boolean Find(int target, int [][] array) {
        int i = array.length-1;
        int j = 0;
        while(i >= 0 && j < array[0].length){
            int temp = array[i][j];
            if(temp == target) return true;
            else if(temp < target) j++;
            else i--;
        }
        return false;
    }
}
运行时间：210ms
占用内存：18792k
Idea 1
左下角元素为此列最大，此行最小
